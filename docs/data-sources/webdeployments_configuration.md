# genesyscloud_webdeployments_configuration (Data Source)

Data source for Genesys Cloud Web Deployments Configurations. Select a configuration by name.

## Permissions and Scopes

The following permissions are required to use this resource:

* `webDeployments:configuration:add`
* `webDeployments:configuration:delete`
* `webDeployments:configuration:edit`
* `webDeployments:configuration:view`

The following OAuth scopes are required to use this resource:

* `webdeployments`
* `webdeployments:readonly`
