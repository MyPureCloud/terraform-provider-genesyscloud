## Example Usage

```terraform
resource "genesyscloud_architect_datatable" "customers" {
  name        = "Customers"
  division_id = genesyscloud_auth_division.home.id
  description = "Table of Customers"

    name  = "key"
    type  = "string"
    title = "Email"

    type    = "boolean"

    default = "false"
  }
```
