# genesyscloud_case_management_caseplan_publish (Resource)

Calls POST /api/v2/casemanagement/caseplans/{caseplanId}/publish. Use depends_on so this runs after stageplan and stepplan resources apply their PATCHes. Increment revision to publish again after later edits. Destroy only removes this from Terraform state (no unpublish API).

## API Usage
The following Genesys Cloud APIs are used by this resource. Ensure your OAuth Client has been granted the necessary scopes and permissions to perform these operations:

* [POST /api/v2/casemanagement/caseplans/{caseplanId}/publish](https://apicentral.genesys.cloud/api-explorer#post-api-v2-casemanagement-caseplans--caseplanId--publish)

# Replace caseplan_id with genesyscloud_case_management_caseplan.<name>.id (or import id).
# Use depends_on = [ ... stageplans, stepplans ... ] so publish runs after configuration PATCHes.
